# Yogesh's Personal Portfolio

A personal portfolio website showcasing my skills, projects, and experience as a Full Stack MERN Developer. Built using HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Mobile-friendly layout that looks great on all devices.
- **Dynamic Navigation**: Active navigation links update automatically on scroll.
- **Projects Section**: Highlights recent projects like *Thumblify*, *SkillUp*, and *VibePlay*.
- **Skills Showcase**: Animated progress bars for technical skills.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Animations, Custom Variables)
- Vanilla JavaScript
- [Boxicons](https://boxicons.com/) (For UI icons)

## Folder Structure
To keep the project organized, files are structured into standard web development folders:

```
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── script.js       # Main Javascript file
├── images/             # All image assets
├── assets/             # Downloadable files (CV, Resume)
└── index.html          # Main HTML entry point
```

## Setup / Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/yogeshhh03/Portfolio-.git
   ```
2. Open the project folder.
3. Simply open `index.html` in your browser, or use an extension like **Live Server** in VS Code for a better development experience.
